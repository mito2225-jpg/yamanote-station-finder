# GitHub リポジトリ接続コマンド

## あなたのGitHubユーザー名を確認して、以下のコマンドを実行してください：

### 1. リモートリポジトリを追加
```bash
git remote add origin https://github.com/YOUR_USERNAME/yamanote-station-finder.git
```

### 2. 最初のプッシュ
```bash
git push -u origin main
```

### 3. 確認
```bash
git remote -v
```

## 📝 注意事項
- `YOUR_USERNAME` を実際のGitHubユーザー名に置き換えてください
- リポジトリ名は `yamanote-station-finder` で作成してください
- Public リポジトリとして作成してください（無料デプロイに必要）

## 🔍 GitHubユーザー名の確認方法
1. GitHub.com にログイン
2. 右上のプロフィール画像をクリック
3. 表示されるメニューでユーザー名を確認
